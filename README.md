**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 健康视频
5. 医生信息
6. 药品列表
7. 健康论坛
8. 健康资讯
9. 留言反馈
10. 个人中心
11. 我的发布
12. 我的收藏
13. 后台管理
14. 在线咨询


**管理员功能：**
1. 登录
